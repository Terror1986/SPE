# SPE ARC-AGI-3 Submission

1. Register at kaggle.com/competitions/arc-prize-2026-arc-agi-3
2. Get API key from three.arcprize.org
3. Add API key to Kaggle notebook secrets as ARC_API_KEY
4. Upload spe_agent.py to Kaggle notebook
5. Submit
